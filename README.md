# sfml-image-viewer
Basic image viewer written with C++ and SFML with basic argument control.

# Commands
## Color control:
```
c-[colorchannel]=[newval]
```
`[colorchannel]` can be `r`, `g`, `b`, or `a` (red, green, blue, or alpha)


## Scale control:
```
s=[newval]
```


## Rect control:
```
r-[direction]=[newval]
```
`[direction]` can be `l`, `t`, `w`, or `h` (left, top, width, or height)
